# 📚 Frontend Exam App

Простое оффлайн-приложение для изучения и повторения frontend-тем в формате карточек с режимом экзамена.

---

## 🚀 Возможности

* ➕ Добавление карточек (вопрос / ответ)
* ✏️ Редактирование карточек
* 🗑️ Удаление карточек
* 🎯 Режим экзамена (случайный порядок)
* 📊 Подсчёт правильных и неправильных ответов
* 🔁 Сброс экзамена (Reset)
* 💾 Автоматическое сохранение в localStorage
* 📱 Полностью работает без интернета

---

## 🧠 Как это работает

* При первом запуске данные загружаются из `cards.json`
* Далее всё хранится в `localStorage`
* Приложение полностью оффлайн
* Все изменения сохраняются автоматически

## 🏗️ Структура проекта

frontend-exam-app
│
├── index.html        # структура страницы
├── style.css         # стили
├── cards.json        # стартовые данные
├── main.js           # точка входа
│
└── js/modules

    └── storage.js    # загрузка данных
